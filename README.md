# 博客精选

## 中文

### 认知科学 / 学习 / 个人成长
- **Mind Hacks**
  - 谁的博客：刘未鹏
  - 推荐理由：
    - 聚焦认知科学、学习方法与程序员思维，《暗时间》等文章广泛流传。
    - 强调元认知，讨论“如何学习”“如何思考”，而不是只教具体技巧。
  - 适合谁：希望系统升级学习方法、思考方式的程序员 / 知识工作者。
  - 博客链接：http://mindhacks.cn （如访问困难，可搜索 GitHub 镜像备份）
 

### **技术 / 创业 / 互联网**

- **阮一峰的网络日志**
  - 谁的博客：阮一峰
  - 推荐理由：
    - 持续十余年的技术与互联网随笔，从 Web、JavaScript、开源生态到时事评论。
    - 是中文世界少见的“计算机 + 互联网史备忘录”，适合系统补基础、看趋势。
  - 适合谁：想建立计算机基础与互联网整体视角的开发者 / 从业者。
  - 博客链接：https://www.ruanyifeng.com/blog/

- **酷壳（CoolShell）**
  - 谁的博客：陈皓（左耳朵耗子）
  - 推荐理由：
    - 极重实战的工程博客，覆盖并发、性能优化、工程文化与职业发展。
    - 「程序员练级攻略」等系列文章影响了一代中文程序员。
  - 适合谁：后端 / 架构 / SRE / 对工程实践和技术职业路径有强需求的人。
  - 博客链接：https://coolshell.cn/

- **当然我在扯淡**
  - 谁的博客：王垠
  - 推荐理由：
    - 深挖编程语言、编译器、操作系统、数学与教育问题。
    - 观点非常强烈，很多文章会直接挑战你对“学习 / 工作”的默认认知。
  - 适合谁：想从底层理解计算机本质与思考方式、能接受强烈个人观点的人。
  - 博客链接：https://www.yinwang.org/

- **小道消息**
  - 谁的博客：冯大辉（Fenng）
  - 推荐理由：
    - 聚焦中国互联网行业的产品、运营、公司与事件，视角偏“圈内”。
    - 风格犀利，对一线从业者有不少参考价值。
  - 适合谁：想理解中国互联网公司、产品与行业演化的从业者。
  - 博客链接：微信公众号「小道消息」

## 英文

### ** 思维模型 / 决策 / 个人成长**

- **Farnam Street**
  - 谁的博客：Shane Parrish
  - 推荐理由：
    - 专注 “Better Thinking, Better Decisions”，从心理学、历史、商业中提炼 mental models。
    - 文章、课程和书系《The Great Mental Models》互相补充。
  - 适合谁：想提升决策质量、构建自己的思维模型库的人。
  - 博客链接：https://fs.blog/

- **James Clear**
  - 谁的博客：James Clear
  - 推荐理由：
    - 围绕《Atomic Habits》展开，讲 1% 复利、习惯设计与行为改变。
    - 文章短小实用，易于快速读完并直接应用。
  - 适合谁：想通过习惯改变现实结果的人。
  - 博客链接：https://jamesclear.com/

- **Study Hacks**
  - 谁的博客：Cal Newport
  - 推荐理由：
    - 从学生博客发展而来，聚焦深度工作、知识工作者效率和数字极简主义。
    - 和《Deep Work》《Digital Minimalism》一脉相承。
  - 适合谁：被信息流淹没、想要“深度工作”的学生 / 知识工作者。
  - 博客链接：https://calnewport.com/

- **Seth’s Blog**
  - 谁的博客：Seth Godin
  - 推荐理由：
    - 每天一篇极短博文，持续多年，围绕品牌、信任、创造力与组织文化。
    - 训练你用极简语言思考价值与营销。
  - 适合谁：想锻炼“用一句话说清价值”的产品人 / 市场人。
  - 博客链接：https://seths.blog/

- **The Marginalian**
  - 谁的博客：Maria Popova
  - 推荐理由：
    - 以文学、哲学和科学作品为素材的长文书评 / 随笔。
    - 适合慢读，用来沉淀对意义、情感、时间等主题的思考。
  - 适合谁：喜欢通过经典作品反思人生意义与情感的人。
  - 博客链接：https://www.themarginalian.org/

### ** Build / 科技 / 投资**

- **Essays（Paul Graham）**
  - 谁的博客：Paul Graham（Y Combinator 联合创始人）
  - 推荐理由：
    - 《How to Do Great Work》等文章被全球创业者反复引用。
    - 讨论创业、黑客文化、写作与“如何做伟大工作”，思想密度很高。
  - 适合谁：想创业，或想提升自己“做事品味”的程序员 / 创始人。
  - 博客链接：https://www.paulgraham.com/

- **Stratechery**
  - 谁的博客：Ben Thompson
  - 推荐理由：
    - 聚焦互联网公司的商业与战略（Apple、Meta、字节、媒体平台等）。
    - 分析产品、商业模式、媒介与监管，是理解科技商业逻辑的经典入口。
  - 适合谁：想系统理解“产品 × 商业模式 × 战略”的从业者。
  - 博客链接：https://stratechery.com/

- **Gates Notes**
  - 谁的博客：Bill Gates
  - 推荐理由：
    - 写科技、气候变化、传染病防治与年度书单，从“世界问题”视角看技术与公益。
    - 能看到一个技术富豪如何思考全球议题与长期投资。
  - 适合谁：对全球议题、科技与公益交汇点感兴趣的人。
  - 博客链接：https://www.gatesnotes.com/

- **Naval（nav.al）**
  - 谁的博客：Naval Ravikant
  - 推荐理由：
    - 极简文字 + 播客形式，围绕创业、投资、财富与幸福。
    - 提出 “Leverage”“Specific Knowledge”“Judgment” 等概念，被大量引用。
  - 适合谁：想在“如何获得财富与自由”上形成自己哲学的人。
  - 博客链接：https://nav.al/

- **Elad Blog**
  - 谁的博客：Elad Gil
  - 推荐理由：
    - 讨论高增长公司的扩张、融资、组织设计等，很多内容是《High Growth Handbook》的延伸。
    - 偏“已经起飞的公司如何继续爬坡”的视角。
  - 适合谁：已在或想进入高速增长创业公司的创始人 / 高管。
  - 博客链接：https://eladgil.com/

- **Tim Ferriss Blog**
  - 谁的博客：Tim Ferriss
  - 推荐理由：
    - 以自我实验著称，围绕效率、习惯、lifestyle design 展开。
    - 常有对世界级选手（投资人、运动员、创业者）的长访谈拆解。
  - 适合谁：想系统调整工作方式、重新设计生活结构的人。
  - 博客链接：https://tim.blog/

### ** 经济学 / 社会科学 / 理性思考**

- **Marginal Revolution**
  - 谁的博客：Tyler Cowen & Alex Tabarrok
  - 推荐理由：
    - 历史悠久的经济学博客，内容覆盖经济学、政策、文化、旅行等。
    - 更新频率很高，可以当成“经济学视角的世界观察日志”。
  - 适合谁：对用经济学视角理解世界感兴趣的人。
  - 博客链接：https://marginalrevolution.com/

- **Astral Codex Ten**
  - 谁的博客：Scott Alexander
  - 推荐理由：
    - 极长篇、信息密度极高，结合心理学、医学、统计与哲学讨论社会议题。
    - 是理性主义 / effective altruism 圈的一个重要节点。
  - 适合谁：愿意读超长文章、希望在复杂议题上训练理性思维的人。
  - 博客链接：https://www.astralcodexten.com/

- **Wait But Why**
  - 谁的博客：Tim Urban
  - 推荐理由：
    - 用长文 + 小人画解释 AI、费米悖论、拖延症、人生规划等复杂问题。
    - 既轻松好读，又能逼你认真思考“时间 / 人生 / 技术”的长期问题。
  - 适合谁：想用轻松方式理解深刻话题、接受长文阅读的人。
  - 博客链接：https://waitbutwhy.com/
